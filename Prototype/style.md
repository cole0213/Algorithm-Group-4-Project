# 스타일 가이드

## 컨셉

- 인사담당자가 장시간 사용하는 도구 → 눈의 피로를 줄이는 차분한 색조
- 정보 밀도가 높은 UI → 여백과 구분선으로 가독성 확보
- 하이라이트·유사문장 색상이 핵심 → 배경색과 충분한 대비 유지

## 레이아웃

### 전체 구조

| 영역 | 너비 |
| ---- | ---- |
| 상단 바 | 100%, 높이 52px |
| 좌측 리스트 | 220px 고정 |
| 우측 포트폴리오 영역 | 나머지 전체 (flex) |

### 포트폴리오 패널

패널은 플로팅 박스 방식이 아닌 **문서 뷰어** 형식. 박스 테두리·그림자·모서리 없음.

| 항목 | 값 |
| ---- | -- |
| 너비 | `flex: 1` 동적 분할 (1개=100%, 2개=50%, 3개=33%, 4개=25%) |
| 최대 패널 수 | 4개 |
| 패널 간 구분 | `border-right: 1px solid #DEE2E6` 세로선만 사용 |
| 내부 패딩 | 20px 24px (상하 / 좌우) |
| 모서리 둥글기 | 없음 |
| 그림자 | 없음 |

### 패널 탭 (상단)

| 항목 | 값 |
| ---- | -- |
| 높이 | 38px |
| 하단 보더 | 2px solid — 지원자별 고유 색상 (accent color) |
| 배경 | `#FFFFFF` |
| 탭 내용 | 지원자 라벨 · 이름 + 매칭률 뱃지 + [X] 닫기 버튼 |

## 컴포넌트

### 지원자 리스트 항목

```
┌──────────────────────┐
│ 지원자C         80%  │  ← 선택 시 좌측 파란 테두리 + 배경 #EEF2FF
│ React · Python · AWS │  ← 보조 텍스트로 주요 기술 미리보기
└──────────────────────┘
```
- 기본 배경: 투명
- 호버: `#F1F3F5`
- 선택: `#EEF2FF`, 좌측 보더 3px `#4361EE`

### 기술스택 뱃지

```
 [React]   [Python]   Vue   Docker
```
- 필요스펙 매칭: 배경 `#FFD43B`, 텍스트 `#5C4400`, 보더 없음
- 미매칭: 배경 `#F1F3F5`, 텍스트 `#6C757D`
- 패딩: `2px 8px`, 둥글기: `4px`

### 매칭률 표시

```
 ● 80%   ← 색상은 매칭률 색상 기준 적용
```

### md 텍스트 렌더링

포트폴리오 본문은 md 형식 텍스트로 표시.

| 요소 | 크기 | 굵기 | 색상 |
| ---- | ---- | ---- | ---- |
| `# 제목` (H1) | 20px | 700 | `#1A1A2E`, 하단 보더 2px `#DEE2E6` |
| `## 섹션` (H2) | 14px | 700 | `#4361EE`, uppercase |
| `### 소제목` (H3) | 13.5px | 600 | `#1A1A2E` |
| 본문 paragraph | 13.5px | 400 | `#343A40` |
| 목록 항목 | 13.5px | 400 | `#343A40`, `–` 앞머리 |

### 동기화 스크롤 토글

| 상태 | 배경색 | 노브 위치 |
| ---- | ------ | --------- |
| OFF | `#CED4DA` | 좌측 |
| ON | `#4361EE` | 우측 |
| 전환 애니메이션 | `background 0.2s, left 0.2s` | |

- 위치: 우측 포트폴리오 툴바 우측 끝 고정
- 크기: 34px × 19px, 노브 15px × 15px

### 원본 링크 버튼

- 배경: `#4361EE`, 텍스트: 흰색
- 호버: `#3A50D9`
- 크기: inline, padding `7px 18px`, 둥글기 6px
- 위치: 포트폴리오 본문 하단

### 설정 드로어

- 오른쪽에서 슬라이드 인 (width: 240px)
- 배경: `#FFFFFF`, 좌측 보더 1px `#DEE2E6`
- 각 토글: 레이블 + 오른쪽 토글 스위치

## 인터랙션

| 상태 | 동작 |
| ---- | ---- |
| 리스트 항목 클릭 | 우측에 포트폴리오 패널 추가 |
| 패널 [X] 클릭 | 해당 패널 제거, 리스트 항목 선택 해제 |
| 동기화 스크롤 토글 | 즉시 적용, 패널 간 스크롤 위치 동기화/분리 |
| 유사문장 숨기기 | 해당 구간 opacity 0.35, 색상 회색으로 전환 (애니메이션 200ms) |
| 검색 입력 | 실시간 필터링, 미보유 지원자 opacity 0.4로 흐림 처리 |
| ⚙ 버튼 클릭 | 설정 드로어 슬라이드 인 (애니메이션 250ms) |
