## 만든 것들

### 오늘뭐먹 (개인 프로젝트)

가장 최근에 만든 앱이에요. 앱스토어에 배포도 했습니다.

![오늘뭐먹 앱 스크린샷1](https://images.unsplash.com/photo-1512621776951-a57141f2eefd?w=400)
![오늘뭐먹 앱 스크린샷2](https://images.unsplash.com/photo-1546069901-ba9599a7e63c?w=400)

**만든 이유**
매일 점심 메뉴 고르는 게 너무 힘들어서요. 친구들이랑 밥 먹을 때 항상 "아무거나"→ "아무거나는 아무거나고" 싸움이 반복되는 걸 보고 만들었습니다.

**주요 기능**
- 혼자 먹을 때: 룰렛 돌려서 메뉴 랜덤 선택
- 같이 먹을 때: 방 만들고 코드 공유, 각자 싫어하는 음식 제외하고 공통 추천
- 즐겨찾기 음식점 등록, 위치 기반 주변 식당 연동 (카카오맵 API)
- 먹은 음식 히스토리 (같은 거 너무 자주 추천 안 하게)

**기술적으로 신경 쓴 부분**
여러 명이 같은 방에서 동시에 선택할 때 실시간 동기화가 필요했는데 Firestore 실시간 리스너로 처리했습니다. 처음에 snapshot listener 해제를 제대로 안 해서 메모리 누수가 났었는데 deinit에서 확실히 해제하도록 고쳤어요.

```swift
class RoomViewModel: ObservableObject {
    @Published var participants: [Participant] = []
    private var listener: ListenerRegistration?

    func observeRoom(roomId: String) {
        listener = Firestore.firestore()
            .collection("rooms")
            .document(roomId)
            .collection("participants")
            .addSnapshotListener { [weak self] snapshot, error in
                guard let docs = snapshot?.documents else { return }
                self?.participants = docs.compactMap {
                    try? $0.data(as: Participant.self)
                }
            }
    }

    deinit {
        listener?.remove()  // 꼭 해제해줘야 합니다
    }
}
```
