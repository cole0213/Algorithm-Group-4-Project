3. 오픈소스: go-circuit (개인 프로젝트)
   ──────────────────────────────────────
   Go용 경량 Circuit Breaker 라이브러리.

   회사에서 쓰려고 만들었는데 다른 데서 쓸 수도 있을 것 같아서 오픈소스로 공개했어요.
   상태 전이(Closed/Open/Half-Open)를 명확하게 추적하고 메트릭 수집 인터페이스 제공.

   GitHub: github.com/taeyang-k/go-circuit
   Stars: 340+ (생각보다 많이 써주셔서 감사합니다)

   ```go
   cb := circuit.New(circuit.Config{
       MaxFailures:  5,
       Timeout:      30 * time.Second,
       HalfOpenMax:  2,
       OnStateChange: func(from, to circuit.State) {
           metrics.RecordTransition(from.String(), to.String())
       },
   })

   result, err := cb.Execute(ctx, func() (any, error) {
       return callExternalService()
   })
   ```
