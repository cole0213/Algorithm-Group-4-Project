1. 링 버퍼 기반 무잠금 큐 라이브러리 (개인 오픈소스)
---------------------------------------------------------------------------

실무에서 쓸 만한 임베디드용 단일생산자-단일소비자 무잠금 큐가 없어서 직접 만들었습니다.

특징:
  - C99 기준, 외부 의존성 없음
  - 메모리 배리어 명시적 삽입 (GCC __sync / C11 atomics 지원)
  - 단일 생산자 / 단일 소비자 환경 전제
  - ARM, x86 테스트 완료

GitHub: github.com/seokho-yun/lockfree-rbuf
Star: 120+

주요 코드:

    /* 생산자 측 삽입 */
    bool rbuf_push(rbuf_t *q, const void *data, size_t len)
    {
        uint32_t head = atomic_load_explicit(&q->head, memory_order_relaxed);
        uint32_t next = (head + 1) & q->mask;

        if (next == atomic_load_explicit(&q->tail, memory_order_acquire))
            return false;  /* full */

        memcpy(q->buf + head * q->elem_size, data, len);
        atomic_store_explicit(&q->head, next, memory_order_release);
        return true;
    }

2. CANalyzer 로그 파서 (사내 도구)
---------------------------------------------------------------------------

CANalyzer에서 출력되는 .asc 로그 파일을 파싱해서
DBC 파일 기반으로 신호 값을 디코딩하고 CSV로 변환하는 도구.

Python으로 짰습니다. 3일 만에 만들었는데 팀에서 지금도 씁니다.
GUI 붙여달라는 요청이 있었지만 거절했습니다. CLI로 충분합니다.

사용 예시:

    $ python can_parser.py --asc log_20260310.asc --dbc vehicle.dbc --out signals.csv
    Parsed 1,204,381 frames in 3.2s
    Decoded signals: 48 (from 12 messages)
    Output: signals.csv
