## 프로젝트

### 1. WaveDash - 실시간 분석 대시보드

![WaveDash 스크린샷](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=800)

회사 프로젝트입니다. B2B 마케팅 분석 SaaS예요.

**뭘 만들었나요?**

기존에 Chart.js로 만들어진 스파게티 코드 대시보드를 처음부터 다시 만들었어요. 제가 주도해서 기술 스택 선정부터 컴포넌트 설계까지 했습니다.

**주요 특징**
- WebSocket으로 1초마다 데이터 갱신
- 커스텀 훅으로 차트 로직 분리해서 테스트 가능한 구조로
- Virtualized List로 수만 건 데이터 렌더링 최적화

```tsx
// 커스텀 훅 예시
function useRealtimeMetrics(channelId: string) {
  const [metrics, setMetrics] = useState<Metric[]>([]);

  useEffect(() => {
    const ws = new WebSocket(`wss://api.wavedash.io/metrics/${channelId}`);
    ws.onmessage = (e) => setMetrics(JSON.parse(e.data));
    return () => ws.close();
  }, [channelId]);

  return metrics;
}
```

**사용 기술**: React, TypeScript, Recharts, Zustand, WebSocket, Vite
